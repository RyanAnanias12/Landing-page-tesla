## Descrição

Recriação de landing page no estilo Tesla, apresentando o robô humanóide **Optimus Gen 2** com animações GSAP. Desenvolvida como exercício prático de front-end a partir de uma aula do canal [codeboost](https://www.youtube.com/watch?v=mmPj7O0jcIk&t=2218s) no YouTube.

## Stack

| | Tecnologia | Uso |
|---|---|---|
| 🔴 | HTML5 | Estrutura semântica |
| 🟣 | SCSS | Estilização modular (fonte de verdade) |
| 🔵 | CSS3 | Compilado do SCSS |
| 🟡 | JavaScript | Interatividade |
| 🟢 | GSAP 3 | Animações de entrada suaves |
| ⚪ | Inter · Plus Jakarta Sans | Tipografia |

---

## Estrutura

```
Landing-page-tesla/
├── css/
│   └── main.min.css          ← compilado do SCSS
├── scss/
│   └── *.scss                ← fonte de verdade dos estilos
├── js/
│   └── scripts.js            ← animações GSAP
├── img/
│   ├── ilustra-hero.png
│   ├── logo.svg
│   └── *.svg · *.png
└── index (3).html            ← entry point
```

---

## Como rodar

```bash
# clone o repositório
git clone https://github.com/RyanAnanias12/Landing-page-tesla.git

# acesse a pasta
cd Landing-page-tesla

# abra no navegador — sem build necessário
open "index (3).html"
```

> Sem dependências de build. Basta abrir o HTML no navegador.
